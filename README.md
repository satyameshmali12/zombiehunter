## Controls
keys

```bash
  SPACE - JUMP
```
```bash
  F - ATTACK
```
```bash
  T - THROW HAMMER
```
```bash
  LEFT ARROW - MOVE LEFT
```
```bash
  RIGHT ARROW - MOVE RIGHT
```
```bash
  LEFT ARROW + F - MOVE LEFT DOING ATTACK 
```
```bash
  RIGHT ARROW + F - MOVE RIGHT DOING ATTACK 
```
```bash
  SPACE + ATTACK - JUMP HIGH
```
```bash
  Q+U+I+T - QUIT THE GAME
```
